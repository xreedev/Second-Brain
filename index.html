Getting Started

Run it locally

Up and running in four steps. No exotic dependencies required.

01
Clone the repository
Pull the code to your local machine.
git clone https://github.com/xreedev/Second-Brain.git cd Second-Brain
02
Set up the backend
Create a virtual environment and install Python dependencies.
cd backend python -m venv venv source venv/bin/activate # Windows: venv\Scripts\activate pip install -r requirements.txt
03
Configure environment
Create a .env file in the backend directory and set your secrets.
SECRET_KEY=your_secret_key_here DATABASE_URL=your_database_url_here JWT_EXPIRY=3600
04
Start both servers
Run the backend, then open the frontend in your browser.
# Backend python app.py # → http://localhost:5000 # Frontend (new terminal) cd ../frontend python -m http.server 3000 # → http://localhost:3000
